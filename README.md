# DevOps Capstone Project

Customer Accounts Microservice built using Flask and Flask-RESTX.

## Build Status

![Build Status](https://github.com/khalilhamdi13/devops-capstone-project/actions/workflows/ci-build.yaml/badge.svg)

## Description

This project implements a RESTful microservice for managing customer accounts.

### Features

- Create an account
- List all accounts
- Read an account
- Update an account
- Delete an account

## Technologies

- Python
- Flask
- Flask-RESTX
- Flask-CORS
- Flask-Talisman
- Pytest
- GitHub Actions
- Docker
- Kubernetes
